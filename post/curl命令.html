curl命令

Posted on August 25, 2017

CURL

不显示进度条-s，-silent，

curl -s 127.0.0.1:60081/phpfpm

设置请求头信息

curl -H 'Accept-Language: zh' http://a.com

###查看响应头信息 curl -I http://www.baidu.com

指定请求方式

–data/-d 方式指定使用POST方式传递数据

curl -XPUT 'http://es1:9200/t1/users/1'  -d @1.json

请求vhost

curl -H 'Host: test.com' http://127.0.0.1:8000/ops/chk/status.html

传递参数

指定source ip 请求第三方地址，以缓解65k问题

curl -v3 --interface "122.192.68.56" 'http://app-ali.youguoquan.com/init?hunter'

发送与接收压缩的请求

curl -H "Accept-Encoding: gzip"  https://appv2.youguoquan.com/init |gunzip
curl 'http://ip.cn/' -H 'Host: ip.cn' -H 'User-Agent: Mozilla/5.0 (Macintosh; Intel Mac OS X 10.12; rv:54.0) Gecko/20100101 Firefox/54.0'

断点续传 -C

-o: 将文件保存为命令行中指定的文件名到本地 -O: 使用url中默认的文件名保存文件到本地

curl -C -o 1.zip http://www.a.com/1.zip

下载限速 –limit-rate

curl --limit-rate 1000k -o a.zip http://www.a.com/1.zip

忽略证书错误

curl -k https://www.a.com

获取重定向后的页面

curl -L http://www.a.com

授权

curl -u username:password http://www.a.com

设置代理

curl -x proxyserver.com:8080 http://www.example.com

ftp操作

# 列出指定目录下的所有文件
curl -u ftpuser:ftppw -O ftp://ftp_server/public_html/
# 上传多个文件
curl -u ftpuser:ftppw -T "{myfile1.txt, myfile2.txt}" ftp://ftp_server/public_html/
# 从标准输入获取内容保存到服务器的指定文件中
curl -u ftpuser:ftppw -T - ftp://ftp_server/public_html/1.txt
