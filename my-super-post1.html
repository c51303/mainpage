操作Perl数组: shift, unshift, push, pop

Date 2016-02-14(日) Tags pop / shift / unshift / push

pop

pop函数会删除并返回数组的最后一个元素。 在第一个例子（3元素数组）中可以看到，pop函数删除最后一个元素（下标最大）并返回它。

my @names = ('Foo', 'Bar', 'Baz');
my $last_one = pop @names;

print "$last_one\n";  # Baz
print "@names\n";     # Foo Bar

push

push 函数可以在数组的后面添加一个或多个值。(当然，也可以添加0个值，但是没有用，不是么？)

my @names = ('Foo', 'Bar');
push @names, 'Moo';
print "@names\n";     # Foo Bar Moo

my @others = ('Darth', 'Vader');
push @names, @others;
print "@names\n";     # Foo Bar Moo Darth Vader

在这个例子中我们最初有个两元素数组。之后我们向数组尾部push了一个标量，数组扩展成了3元素数组。 第二次调用push，我们push了@others数组的内容到@names尾部，把它扩充成5元素数组。
